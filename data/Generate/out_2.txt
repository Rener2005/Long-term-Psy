# 第一次咨询

## 用户背景
[基本信息]  
C 同学，男，大三学生。C 同学的母亲在家务农，父亲常年外出务工，家庭条件较为困难，家庭教育存在一定缺失，未接受过相关的性教育。上大学后未加入任何团学组织或社团，课余时间基本待在宿舍，生活较为单调，社交圈较小，自律意识不强，易做出冲动行为。

[近期生活事件]  
C 同学假扮女生在网上与本校女生 Y 同学聊亲密话题，并在见面后有不良行为。事件被曝光至非官方自媒体平台，引发重大校园舆情，后被学校相关部门介入调查并对 C 同学给予处分。

[心理状态]  
C 同学感到内心痛苦，对自己的行为感到反思和悔恨，但也伴随自卑和焦虑情绪，难以正确看待自身行为及其后果。

[此次咨询的目的]  
帮助 C 同学正确看待自己过去的违纪行为，并接纳受处分的事实，同时纠正认知偏差，提升自信心，帮助其顺利回归正常的学习和生活。

## 咨询策略

[咨询目标]  
帮助 C 同学正确看待其违纪行为及处分事实，尽可能降低该事件对其心理和情绪的长期影响，并协助其重建自信和改善社交能力。

[咨询计划]  
通过认知行为疗法，引导 C 同学正确看待违纪行为，并接纳事实，通过放松训练和正念冥想的方式帮助其缓解焦虑，提升自信心。在咨询过程中，帮助其辨识并纠正认知偏差，以改善情绪和行为。
